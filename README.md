# Sprint_7

test_create_courier - тест на создание курьера (позитивный)
test_create_courier_create_two_courier_error - проверяем, что нельзя создать двух одинаковых курьеров
test_create_courier_no_login_error - проверяем, что нельзя создать курьера без логина
test_create_courier_no_password_error - проверяем, что нельзя создать курьера без пароля
test_create_orders - проверяем создание заказа с использованием параметразации, где:
    1. Передаем Color BLACK
    2. Передаем Color GREY
    3. Передаем одновременно два значения поля Color
    4, Не передаем Color
test_get_orders - проверяем метод получения заказов, провермя что в ответе есть заказы
test_login_courier - курьер может авторизоваться
test_login_courier_no_login_error - проверяем. что нельзя авторизоваться без логина
test_login_courier_no_password_error - проверяем, что нельзя авторизоваться без пароля
test_login_courier_no_exist_courier_error - проверяем, что нельзя авторизваться с несуществующим логином
test_login_courier_wrong_password_error - проверяем, что нельзя авторизоваться с неправильным паролем
